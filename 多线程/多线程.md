
###如何停止线程
1、interrupt:设个标志
2、isInterrupted:只看不擦
3、interrupted:看完就擦,只能问自己

// 示例
Thread t = new Thread(() -> {
    // 第一次调用 interrupted()
    System.out.println(Thread.interrupted());  // false（初始为 false）
    
    Thread.currentThread().interrupt();        // 设置中断标志
    
    System.out.println(Thread.interrupted());  // true（返回 true，然后清除标志）
    System.out.println(Thread.interrupted());  // false（已被清除）
    
    System.out.println(Thread.currentThread().isInterrupted()); // false
});
t.start();
